# 抖音古言大女主漫剧开篇法则

## 核心原则：前5秒给冲突，前30秒给承诺

面向31-40岁白领女性，开篇必须让观众立刻知道：女主被夺走了什么、她是否清醒、她下一步能不能赢。

**黄金5秒：**
- 第1秒：古风强画面，如喜堂、宫门、血书、和离书、圣旨、药盏、凤冠。
- 第3秒：核心压迫出现，如背叛、诬陷、夺权、赐死、替嫁、断亲。
- 第5秒：女主给出清醒反应，如冷笑、反问、亮证据、递文书、藏后手。

**前30秒必须完成：**
1. 建立女主处境：被夫家、家族、朝堂、命运或旧爱压迫。
2. 展示女主不是弱者：她看清局势，并有第一步动作。
3. 抛出追看问题：她为何早有准备？谁会翻车？她能拿回什么？

## 6种古言开场模板

### 模板一：和离当场型

适用：和离改命、重生复仇、宅斗掌家。

结构：
1. 夫君/婆母/外室当众羞辱女主。
2. 女主不哭不闹，拿出和离书或嫁妆单。
3. 她指出一个足以反转局面的证据。

示例：

```markdown
△ （全景）侯府正厅，红绸未撤，外室已被扶上主位。

**婆母**（冷声）："你无所出，正妻之位也该让了。"

△ （特写）女主指尖按住一纸和离书，朱印鲜红。

**女主**（抬眼）："让位可以。先把我母亲留下的三十七箱嫁妆，一箱不少地抬出来。"
```

### 模板二：重生节点型

适用：重生复仇、穿书觉醒、真假千金。

结构：
1. 女主从前世惨局/原剧情死亡点醒来。
2. 立刻回到关键事件前。
3. 她做出和前世不同的选择。

示例：

```markdown
△ （特写）雨水混着血，从女主指尖滑落。

△ 她猛地睁眼。喜烛燃得正旺，门外传来迎亲锣鼓。

**婢女**（急声）："小姐，二姑娘又说身子不适，要您替她上花轿。"

△ （特写）女主看向铜镜，慢慢笑了。

**女主**："好啊。这一次，该换他们入局了。"
```

### 模板三：当众诬陷型

适用：宅斗掌家、宫廷权谋、女官逆袭。

结构：
1. 女主被当众扣上罪名。
2. 反派胜券在握。
3. 女主用证据或规则反击。

示例：

```markdown
△ （全景）祠堂内，族老列坐，女主跪在堂前。

**继母**（掩泪）："账房短了三千两，只有她碰过钥匙。"

△ （特写）女主从袖中抽出另一册账本。

**女主**："母亲说的是这把钥匙，还是您昨夜交给账房的那把？"
```

### 模板四：圣旨命运型

适用：宫廷权谋、女官逆袭、神女归位。

结构：
1. 圣旨/凤印/官令降临，女主命运被公开裁决。
2. 所有人以为她完了。
3. 圣旨内容或女主反应反转。

示例：

```markdown
△ （全景）宫门大开，传旨太监踏雪而入。

**太监**："沈氏女，接旨。"

△ 众人跪下，只有女主抬起头。

**反派**（低声）："这回，看谁还救得了你。"

△ （特写）明黄圣旨展开，第一行字映入女主眼底。
```

### 模板五：替嫁入局型

适用：替嫁翻身、权谋同盟。

结构：
1. 女主被推上花轿或替人入府。
2. 入府即遭审问/下马威。
3. 女主识破婚局背后的真正目的。

示例：

```markdown
△ （中景）红盖头被一把挑开，满室烛火冷得像刀。

**男主**："你不是我要娶的人。"

△ （特写）女主看向桌上未干的药渍。

**女主**："巧了。你也不是病弱到不能下床的人。"
```

### 模板六：神格觉醒型

适用：仙侠神女、宿命复仇。

结构：
1. 女主被剜骨、献祭、退婚或贬落。
2. 旧身份被否定。
3. 神印/命格/法器觉醒，抛出真实身份。

示例：

```markdown
△ （全景）祭台上，风雪倒卷。

**反派**："她不过是个替命的凡女。"

△ （特写）女主腕间裂开一道金纹。

△ 断剑嗡鸣，万盏天灯同时熄灭。

**女主**（低声）："凡女？你们跪错人了。"
```

## 开场自检

- 是否5秒内出现强冲突和古风强画面？
- 女主是否在30秒内展现清醒反应？
- 是否避免大段旁白和复杂设定解释？
- 是否让观众明确期待“她下一步怎么赢”？
- 男主是否没有抢走女主的主导权？
