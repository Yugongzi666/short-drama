# 抖音女频古言大女主漫剧题材指南

## 赛道定位

默认创作对象为抖音平台女频漫剧，主攻 **31-40岁白领女性**。她们看古言大女主不是为了单纯少女幻想，而是为了在古代权力场中获得“我清醒、我能赢、我值得被偏爱”的情绪代偿。

核心体验：

- **清醒感：** 女主不恋爱脑，不把婚姻、男人、家族评价当作唯一价值。
- **体面反击：** 不靠撒泼和粗暴复仇，而靠证据、布局、话术、资源和时机。
- **权力成长：** 从被安排、被误解、被夺走，到掌握命运、家业、朝局或名望。
- **轻熟情感：** 男主提供尊重、偏爱、托底和势均力敌的拉扯，不是拯救者。
- **古风审美：** 礼制、场景、道具要精致可视化，适合漫剧分镜呈现；角色衣饰、发型发饰遵循既定参考图，不在剧本中描述。

## 受众画像

| 维度 | 特征 | 创作响应 |
|------|------|----------|
| 年龄 | 31-40岁 | 减少幼稚误会，增加现实处境、婚姻、家族、事业/权力议题 |
| 职业 | 白领/管理岗/专业岗 | 女主必须有能力、判断力和解决问题的手段 |
| 情绪 | 白天克制体面，晚上需要高效解压 | 前5秒给冲突，前3集完成一次反击 |
| 审美 | 轻熟高级、古风精致、强画面 | 避免土味口号、廉价炫富、低幼画风 |
| 价值观 | 要爱情，但更要尊严和主动权 | 男主是同盟和奖励，不是女主人生答案 |

## 子赛道选择

### 1. 重生复仇

- **一句话描述：** 女主带着前世记忆归来，改写被害、被夺、被弃的命运。
- **核心爽点：** 先知布局、证据反杀、恶人自食其果、身份/名誉回收。
- **适合开局：** 前世惨死或大婚当日被背叛，下一秒重回关键节点。
- **视觉钩子：** 雨夜血迹、断簪、火烧喜堂、重睁双眼、铜镜裂纹。
- **避雷：** 不要让女主只靠记忆开挂；每次翻盘都要有行动成本。

### 2. 和离改命

- **一句话描述：** 被夫家轻贱的女子主动和离，夺回嫁妆、名声和人生。
- **核心爽点：** 休夫反转、夺回嫁妆、婆家翻车、前夫追悔莫及。
- **适合开局：** 夫君带外室登门、婆母逼让正妻位、女主当众递盖印文书。
- **视觉钩子：** 盖印文书、嫁妆箱牌、碎玉镯、祠堂对峙、女主推门离开。文字不入镜。
- **避雷：** 不要把女主胜利写成“换个更强男人接盘”；重点是自我重建。

### 3. 替嫁翻身

- **一句话描述：** 女主被迫替嫁到险局，却借婚局进入更高权力场。
- **核心爽点：** 被低估、识破阴谋、掌家立威、假夫妻变同盟。
- **适合开局：** 花轿临门、妹妹逃婚、女主被推上轿，入府即遭下马威。
- **视觉钩子：** 盖头落地、冷府大门、地面血滴、洞房审问、嫁妆箱暗格。
- **避雷：** 男主可以强势，但不能羞辱和控制女主来制造爱情。

### 4. 宅斗掌家

- **一句话描述：** 女主在家族内宅中从被压制者成长为掌局者。
- **核心爽点：** 管账夺权、识破栽赃、打脸恶亲、重整家业。
- **适合开局：** 庶女/继女被诬陷，女主用账本、证人或礼法当场翻盘。
- **视觉钩子：** 合起的账册、钥匙串、掌家印、雪夜跪祠堂、满院灯笼。账册文字不入镜。
- **避雷：** 宅斗要有利益逻辑，不要只写女人互害。

### 5. 宫廷权谋

- **一句话描述：** 女主在皇权棋局中保命、破局、掌权。
- **核心爽点：** 礼法反杀、朝堂借力、后宫设局、身份揭露、权力加冕。
- **适合开局：** 入宫第一夜被赐死/栽赃，女主抓住制度漏洞反转。
- **视觉钩子：** 卷轴、凤印、药盏、宫门雪、烛影屏风、金殿对峙。卷轴文字不入镜。
- **避雷：** 不要美化极端宫斗手段；女主的胜利要有正当性。

### 6. 女官/经商逆袭

- **一句话描述：** 女主用专业能力进入公共权力或商业秩序，打破性别限制。
- **核心爽点：** 才能证明、制度破局、商业翻盘、被公开承认。
- **适合开局：** 女主作品/方案被抢，或被嘲“女子不配议事”。
- **视觉钩子：** 算盘、合起的契书、印章、工坊、告示牌轮廓、御前陈词。可读文字不入镜。
- **避雷：** 不要写成现代职场换皮；古代制度阻力要具体。

### 7. 真假千金古言化

- **一句话描述：** 女主被调换身份或被真假亲情裹挟，最终夺回名分和选择权。
- **核心爽点：** 血脉揭露、假千金翻车、亲情审判、身份回收。
- **适合开局：** 认亲宴被羞辱，女主当场拿出证据或主动断亲。
- **视觉钩子：** 玉佩、胎记、旧印纹路、认亲宴、滴血验亲被反转。
- **避雷：** 不要只靠血统高贵赢；女主的能力要先于身份被看见。

### 8. 穿书觉醒

- **一句话描述：** 女主发现自己是恶毒女配/炮灰正妻，主动改写剧情。
- **核心爽点：** 预判剧情、拒绝工具人命运、抢回主线、反套路爱情。
- **适合开局：** 女主醒在被退婚/被赐死/被陷害的剧情节点。
- **视觉钩子：** 纸页燃烧、命格纹路、画面裂痕、女主撕掉盖印文书。避免剧情文字浮现。
- **避雷：** 不要频繁解释系统规则；抖音漫剧要用行动推进。

### 9. 仙侠神女归位

- **一句话描述：** 被贬、被夺神骨或被误认低贱的女主找回神格与权柄。
- **核心爽点：** 神格回归、旧爱追悔、天命反转、三界承认。
- **适合开局：** 女主被剜骨/退婚/献祭，濒死时神印觉醒。
- **视觉钩子：** 神印、断剑、莲火、天阶、祭台、回眸定格。
- **避雷：** 仙侠设定要为情绪服务，不要堆术语和复杂世界观。

## 题材叠加规则

只选 **1 个主赛道 + 1 个副元素**。副元素用于增加新鲜感，不要抢主题。

| 主赛道 | 推荐副元素 | 适合效果 |
|------|------------|----------|
| 重生复仇 | 宅斗掌家 | 复仇更有家族利益链 |
| 和离改命 | 经商逆袭 | 女性自立感更强 |
| 替嫁翻身 | 权谋同盟 | 婚恋拉扯更高级 |
| 宅斗掌家 | 真假千金 | 身份与资源争夺更强 |
| 宫廷权谋 | 神女归位 | 权力和宿命感更强 |
| 女官逆袭 | 重生复仇 | 专业能力和先知布局结合 |
| 穿书觉醒 | 和离改命 | 反套路和清醒感强 |

避免组合：

- 宫廷权谋 + 过重喜剧：削弱权力压迫感。
- 和离改命 + 男主全程托举：削弱大女主成长。
- 仙侠神女 + 宅斗细碎日常：审美层级不统一。
- 真假千金 + 纯雌竞：容易廉价和低俗。

## 抖音选题判断

一个选题进入 /plan 前必须满足：

1. **5秒能讲清冲突：** 她被谁夺走了什么？
2. **3集能给爽点：** 她如何第一次赢回来？
3. **女主有主动权：** 她的选择推动剧情，而不是被事件拖着走。
4. **有视觉符号：** 至少 3 个可反复使用的古风道具或场景。
5. **有情感拉扯：** 男主/同盟与女主势均力敌，尊重她的目标。
6. **有长线权力目标：** 掌家、翻案、夺印、入朝、复仇、归位等。

## 审美要求

| 维度 | 推荐 | 避免 |
|------|------|------|
| 女主 | 清醒、克制、聪明、有风骨 | 傻白甜、哭包、恋爱脑 |
| 男主 | 强但尊重边界，能托底也能放手 | 油腻霸总、强制爱、爹味说教 |
| 反派 | 有利益、有位置、有压迫感 | 只会尖叫辱骂的工具人 |
| 台词 | 短、准、藏锋，有古言质感 | 现代土味口号、长篇解释 |
| 画面 | 卷轴、雨夜、宫门、烛影、玉佩、血迹 | 低幼Q版、廉价大红大紫、杂乱背景、可读文字上屏、具体角色造型描写 |
| 情绪 | 体面反击、冷静翻盘、被偏爱 | 撕逼、撒泼、无底线雌竞 |
